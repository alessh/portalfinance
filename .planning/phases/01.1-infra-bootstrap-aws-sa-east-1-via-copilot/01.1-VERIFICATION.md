## Production-Discovered Hotfixes (preserved for future operators)

Mirrored from STATE.md:

1. PrivateSubnets export shape (single CSV, use `Fn::Split`)
2. Copilot v1.34 `From: env` allowlist gap (use `Fn::ImportValue`)
3. Env-addon nested-stack race (deploy env first, then addon)
4. DB-bridge SSM tag-based IAM (parameters must be tagged `copilot-application` + `copilot-environment`)
5. DB-access ManagedPolicy (workload-level `addons/db-access.yml`, not env-level)
6. RDS-generated password URL hazard (tighten `ExcludeCharacters` + URL-encode in entrypoint)
7. postgres-js URL forwarding (strip `?sslmode` / `?sslrootcert`; configure `ssl` per client)
8. pg-connection-string v2 silent `verify-full` alias (bake CA bundle into runner image)
9. pg-boss v12 no-auto-create queues (`getBoss()` calls `createQueue` for every QUEUES entry on `start()`)
10. Git Bash path mangling (`MSYS_NO_PATHCONV=1` mandatory for AWS CLI calls with `/copilot/...` parameter names on Windows)
