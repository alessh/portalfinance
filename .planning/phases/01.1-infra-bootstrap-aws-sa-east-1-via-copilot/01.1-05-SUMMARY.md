## Deviations from Plan

### Auto-fixed Issues

1. **PrivateSubnets export shape** — Copilot env emits `${App}-${Env}-PrivateSubnets` as a single comma-separated list, not PrivateSubnet1/2. Fix: `Fn::Split` in `rds-postgres.yml`. Commit: `b7b62b4`.
2. **`From: env` allowlist gap** — `VpcId` not in v1.34 allowlist. Fix: `Fn::ImportValue: !Sub '${App}-${Env}-VpcId'`. Commit: `87c414b`.
3. **Env-addon nested-stack race** — ImportValue cannot resolve while parent env is `CREATE_IN_PROGRESS`. Fix: deploy env first, then add addon (`9ba049e` -> `6bcbae9`).
4. **DB-bridge IAM** — Task launch returned AccessDeniedException reading the SSM bridge until the parameters were tagged `copilot-application` + `copilot-environment`. Commit: `8c59340`.
5. **Workload secret IAM** — Env-level `DBAccessPolicy` is not auto-attached per-service in v1.34. Fix: per-service `copilot/<svc>/addons/db-access.yml`. Commit: `3cab3a2`.
6. **Image bloat** — tsup-bundled worker missing runtime deps. Fix: ship pruned prod `node_modules`. Commit: `e2ce7ef`.
7. **Env scoping** — TURNSTILE_* assertion fired on the worker service. Fix: scope to `SERVICE_NAME=web`. Commit: `e05daf4`.

**Total deviations:** 7 auto-fixed (production-discovered AWS/Copilot edge-cases).
**Impact on plan:** Necessary corrections; no scope creep.
