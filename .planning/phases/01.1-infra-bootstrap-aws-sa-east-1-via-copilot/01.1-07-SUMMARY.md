## Deviations from Plan

### Auto-fixed Issues

1. **www edge 503** — `https://www.portalfinance.app` returned a Cloudflare-fronted 503 because the ALB host-header rule only matched the apex. Cert already covered both names (www is a SAN). Fix: widen the alias list. Commit: `98d8e5f`.
2. **Validation script exit-23 on Windows** — `curl ... -o /dev/null` produced spurious exit 23 on Git Bash for Windows even after HTTP 200. Fix: switch to `curl -I -w '%{http_code}' || true` and judge from captured status code. Commit: `3a0ff27`.

**Total deviations:** 2 auto-fixed.
**Impact on plan:** Both necessary for a working production edge and a portable gate script.
