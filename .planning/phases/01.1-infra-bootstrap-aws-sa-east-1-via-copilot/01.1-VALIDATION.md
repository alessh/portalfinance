## Sampling Rate

- **After every task commit:** Run `pnpm test:unit`
- **After every plan wave:** Run `pnpm test:all`
- **Before `/gsd-verify-work`:** Full suite must be green **AND** `bash scripts/validate-phase-01.1.sh` must exit 0 against the deployed `prod` environment
- **Max feedback latency:** 15 s (unit), 90 s (full suite), 60 s (infra-gate)
