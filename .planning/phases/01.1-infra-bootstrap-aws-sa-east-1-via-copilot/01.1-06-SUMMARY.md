## Deviations from Plan

### Auto-fixed Issues

1. **RDS-generated password URL hazard** — `?` characters terminated the URL userinfo component. Fix: tighten `ExcludeCharacters` in addon (Plan 01.1-05) and URL-encode in entrypoint here. Commit: `863d242`.
2. **postgres-js startup GUC rejection** — `?sslmode=require` rode into Postgres as a startup parameter; server rejected it. Fix: strip params from URL, configure `ssl` per client. Commits: `41a3cf1`, `98e0bbb`, `d1404d2`.
3. **pg-connection-string v2 silent verify-full alias** — caused intermittent CA path failures. Fix: bake CA bundle into the runner image. Commit: `41a3cf1`.
4. **pg-boss v12 no-auto-create queues** — workers crashed with "queue not found". Fix: explicit `boss.createQueue` for every QUEUES entry on start. Commit: `0a3d92b`.

**Total deviations:** 4 auto-fixed (TLS + driver-version edge cases discovered post-deploy).
**Impact on plan:** All necessary for a working production DB connection; no scope creep.

## Issues Encountered

- The TLS pivot took multiple iterations because pg-connection-string v2's `sslmode=require -> verify-full` alias masked the missing CA path; resolved by stripping URL params and binding the CA in code.
